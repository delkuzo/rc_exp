Основная цель проекта:

Создание системы разработки интерфейсов на основе UI Kit из Figma, где автор (эксперт по UX/UI/Figma) описывает интерфейсы через запросы, а AI-разработчик (я) верстает их используя готовую систему компонентов.

Роли участников:

- Автор: Эксперт по интерфейсам, UX, Figma - владелец UI Kit и определяет дизайн - AI-разработчик: Senior Front-End Developer - реализует интерфейсы используя систему компонентов
