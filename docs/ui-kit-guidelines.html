Интеграция с Figma:

- MCP сервер: подключение к Figma файлу для получения данных - Точечное обновление: добавление изменений из Figma по запросу автора - Получение компонентов: извлечение кнопок, цветов, типографики и других элементов по требованию - Обновление проекта: добавление новых контролов в UI Kit по конкретному запросу

UI Kit (основа проекта):

- Цветовая палитра: для светлой и темной темы с одинаковыми названиями, но разными значениями - Типографика: система шрифтов и их стилей - Система отступов и размеров: единая система spacing - Контроллы: готовые компоненты, созданные из типографики, цветов и отступов - Набор иконок: (будет подключен позже)
